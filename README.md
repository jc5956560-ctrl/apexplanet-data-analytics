# ApexPlanet Data Analytics Internship

## Task 1: Foundational Setup & Exploratory Data Analysis (EDA)
- **Objective**: Cleaned the NGO dataset and performed EDA using Python.
- **Libraries Used**: Pandas, NumPy, Matplotlib, Seaborn.
- **Key Insights**:
  1. Most NGOs focus on 'Education & Healthcare' initiatives.
  2. 40% of the surveyed NGOs are based in New Delhi.



## Task 2: SQL for Data Extraction
- **Objective**: Loaded the cleaned NGO dataset into an SQLite database and extracted business insights using SQL queries.
- **Key Actions**:
  1. Handled column naming anomalies (fixed trailing spaces and text typos like 'Headquaters').
  2. Implemented advanced SQL data extractions using Common Table Expressions (CTEs/WITH clauses) and filtering mechanisms (`WHERE`, `LIKE`, `GROUP BY`).
- **Key Insights Extracted**: Successfully segmented NGOs and identified targeted clusters operating specifically within the 'Education' and 'Healthcare' sectors.
